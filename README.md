fbxファイルを読み込み、アニメーションさせるためのjavascriptクラスを作ります。

three.jsにあるfbxのloaderでは、アニメーション出来ないfbxファイルがあります。

ということで自前でfbxのparserとanimaterを作ることにします。

描画はthree.jsか、WebGLで行います。
